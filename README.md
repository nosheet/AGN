# AGN – Assistant de Génération de Notes

⚠️ **Prototype / Proof of Concept**

AGN is a browser-based tool designed to assist healthcare professionals in drafting structured care notes using predefined clinical parameters.

This project is currently in a **development and evaluation phase**.

## Purpose

- Support the formulation of care notes in a structured and consistent manner
- Reduce time spent on repetitive documentation tasks
- Improve clarity and readability of clinical notes

## Important Notice

- This tool **does not store, transmit, or log any patient data**
- Users must **not enter identifying patient information** (name, date of birth, etc.)
- Generated text remains under the responsibility of the user

## Technical Notes

- Static web application (HTML / CSS / JavaScript)
- No backend, no database, no analytics
- Designed to run entirely in the browser

## Status

This repository is shared for **demonstration and testing purposes only**.  
It is not a certified medical device and must not be used as-is in production without proper validation and approval.
